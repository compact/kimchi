Class: Body

Body

new Body(options)

Class for astronomical bodies. All spheres for now. Can only be constructed inside kimchi.space.js. Raw data for each Body is stored in /json/kimchi.space.bodies.json for passing into this constructor.
Parameters:
Name Type Description
options Object Options.
name: Required. Displayed to users.
radiusInKm: In km.
rotation: Vector3 of the body's initial Euler rotation.
labelVisibleDistance: How far away the text mesh remains visible.
mesh: Optional. If not given, a Mesh is automatically generated.
translate: Optional. Translate the Mesh, e.g. for orbit.
rotate: Optional. Rotate the Mesh.
texturePath: Optional path to the texture image. Defaults to 'name.jpg'.
